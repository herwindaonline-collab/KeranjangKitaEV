``` ### Cara Membuka di HP/Laptop: 1. **Simpan sebagai `index.html`**: Simpan kode di atas di Notepad atau teks editor apa saja. 2. **Coba langsung**: Klik dua kali file tersebut, ia akan terbuka di browser Anda (Chrome/Safari). 3. **Pindahkan ke HP**: Anda bisa mengirim file `index.html` ini ke HP Anda lewat WhatsApp atau email, lalu klik untuk membukanya. 4. **Deploy Instan**: Jika ingin punya link URL, buka [Netlify Drop](https://app.netlify.com/drop), lalu *drag & drop* file `index.html` Anda ke sana.
